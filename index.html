Welcome to Bus Finder.
This website provides 3 modes of usage.

  1. Search as per Source and Destination:
    Enter Source (eg, Barasat) and Destination (eg, Baruipur) location.
    On correct location entry (i.e., some buses ply through) :
    • If direct route exists, the bus list is shown below and the textbox will glow green.
    • If no direct route exists, then the textboxes will glow black.
    On incorrect location entry, i.e., if source or destination location incorrect or if no bus ply through either of them, then textbox will glow red.
    Both fields needs to be entered to get a result.
    One can also add via for search by toggling the via switch. Then all 3 fields should be entered.
  2. Search a bus route:
    Enter a bus route (eg, L238, Barasat Baruipur) hit search and full route info will be shown, both for up and down. Textbox will glow green.
    If route number or name is incorrect, it will glow red.
  3. Search as per a particular location:
    Enter a location (eg, Baguiati, Tollygunge) hit search and all the buses that run through that location will be shown. Textbox will glow green. You can then search for individual bus to get complete route info.
    If location incorrect or no buses ply from that location, it will glow red.

For every field, there is autocorrect feature, to a certain extent, as well as a list will appear, as you type. You can either choose from the list or type of your own.
Press Reset All button to clear every fields and set it to default.
Press Day/Night button on top right to toggle to Light and Dark Modes.
